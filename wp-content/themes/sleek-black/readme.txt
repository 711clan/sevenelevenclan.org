-----------Description-----------------
---------------------------------------

A clean and professional-looking theme in black. It is search-engine-optimized, gravater-supported, widget-ready, multi-browser-compatible. (Firefox, IE, and Google Chrome.) Comes with full-width page, post pagination, threaded/paginated comments, comments reply function, smooth scrolling, multi-level drop down menu, custom menu, and multiple widgetized area.

-----------Installation-----------------
---------------------------------------

 - Via WordPress Admin
1. From your sites admin, go to Themes > Install Themes
2. In the search box, type 'Sleek Black' and press enter
3. Locate the entry for 'Sleek Black' (there should be only one) and click the 'Install' link
4. When installation is finished, click the 'Activate' link

 - Manual Install
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory, should look like mywebsite.com/wp-content/themes/sleek-black/
2. Go into your Wordpress Admin, navigate to 'Appearance > Themes'
3. Find the Sleek Black listing on this page and click 'Activate'
